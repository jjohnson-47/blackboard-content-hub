Interactive Graph

y = a * sin(b * x + c) + d

This interactive graph demonstrates a sine wave with adjustable parameters:

  • a (amplitude): Controls the height of the wave
  • b (frequency): Controls how many cycles appear
  • c (phase): Shifts the wave horizontally
  • d (offset): Shifts the wave vertically

Use the sliders to adjust the parameters and see how they affect the graph.
